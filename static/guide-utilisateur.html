01 — Introduction

Qu'est-ce que PropGen Premium ?

Orpi PropGen Premium est un outil de creation de propositions de gestion locative concu pour les gestionnaires du reseau Orpi. Il remplace les anciens processus Excel par une interface moderne et intuitive qui genere des propositions PDF de 11 pages en quelques minutes.

⚡ Points forts

  • Apercu en temps reel — Voyez votre proposition se construire au fur et a mesure de la saisie
  • Simulateur fiscal TMI — Calculez les revenus nets du proprietaire selon sa tranche d'imposition
  • 100% local — Aucun serveur, aucun compte requis, vos donnees restent sur votre poste
  • PDF professionnel — 11 pages fideles a la charte Orpi
  • Sauvegarde automatique — Jamais de perte de donnees

Installation

L'installation ne necessite aucun droit administrateur et prend moins d'une minute :

  1. Double-cliquez sur le fichier OrpiPropGen-Setup-1.0.0.exe
  2. Suivez les etapes de l'assistant d'installation (en francais)
  3. Cochez "Lancer Orpi PropGen" a la fin pour demarrer immediatement
💡 Prerequis Le navigateur Google Chrome ou Microsoft Edge doit etre installe sur votre poste. L'application s'ouvre automatiquement dans le navigateur en mode application (sans barre d'adresse).

Lancer l'application

Apres l'installation, un raccourci "Orpi PropGen" est cree sur votre Bureau et dans le menu Demarrer. Double-cliquez dessus pour lancer l'application.

09 — Questions frequentes

Ou sont stockees mes donnees ?

Toutes les donnees sont stockees localement dans votre navigateur (IndexedDB). Aucune information n'est envoyee sur Internet. Vos donnees restent sur votre poste de travail.

Que se passe-t-il si je ferme l'application en cours de saisie ?

Rien n'est perdu. La sauvegarde automatique enregistre vos modifications en continu. A la reouverture, vous retrouverez votre proposition exactement ou vous l'avez laissee.

Comment transferer mes propositions sur un autre ordinateur ?

Utilisez la fonction Exporter sur le premier poste pour telecharger un fichier JSON, puis Importer ce fichier sur le second poste.

Puis-je modifier une proposition finalisee ?

Oui. Cliquez sur une proposition finalisee dans le tableau de bord pour la rouvrir dans le formulaire. Vous pouvez modifier les informations et regenerer le PDF.

Le navigateur affiche une erreur de securite

L'application fonctionne en mode fichier local. Chrome ou Edge peut afficher un avertissement. Le script de lancement configure automatiquement les permissions necessaires. Si le probleme persiste, relancez l'application depuis le raccourci Bureau.

Comment mettre a jour l'application ?

Votre responsable informatique vous fournira une nouvelle version de l'installateur. Lancez-le pour mettre a jour — vos donnees existantes sont conservees.

💡 Besoin d'aide ? Contactez votre responsable d'agence ou l'administrateur informatique en charge de l'outil.
